# Portfolio Website - Fachri Fathan

Modern & minimalist personal portfolio website with dark theme.


## 🎨 Design System

### Colors (Dark Theme)
- Background: `#0A0A0A`
- Cards: `#141414`
- Accent: `#525252`
- Text: `#FAFAFA`

### Typography
- Font: Inter (Google Fonts)

## 🚀 Quick Start

1. Place your profile photo in `assets/images/profile/`
2. Add project screenshots to `assets/images/projects/`
3. Update content in `index.html`
4. Open `index.html` in browser or use local server
