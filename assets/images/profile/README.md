# Profile Images

Place your profile photos here.

## Recommended Specifications

- **Filename:** `avatar.jpg` or `profile.jpg`
- **Size:** 400x400px (square)
- **Format:** JPG or WebP
- **File size:** Under 100KB for optimal loading

## Usage

The main profile image is referenced in `index.html` as:
```html
<img src="assets/images/profile/chr.jpg" alt="Profile Photo">
```
