# Documents

Place your resume, CV, and certificates here.

## Recommended Files

- `resume.pdf` - Your latest resume/CV
- `resume-id.pdf` - Indonesian version (if applicable)
- `resume-en.pdf` - English version (if applicable)
- `certificates/` - Folder for certification files

## Usage

Link to resume in website:
```html
<a href="assets/documents/resume.pdf" download>Download Resume</a>
```
